# MAANV.AI – AI Co-founder Platform

🚀 Where human imagination meets AI

## Overview
MAANV.AI is an AI-powered co-founder designed to act like a real startup partner.

It helps founders move from idea → execution by combining strategy, product thinking, and growth intelligence in one system.

## What It Does
MAANV.AI functions as a multi-role AI co-founder:

- 🧠 Vision & Strategy  
  Helps validate ideas, analyze markets, and refine business models  

- ⚙️ Product Engineering  
  Assists in building MVPs, structuring features, and system design  

- 📈 Growth & Marketing  
  Guides go-to-market strategies, user acquisition, and positioning  

- 💰 Pitch & Fundraising  
  Helps create pitch decks, financial models, and investor strategies  

## Tech Stack
- Python (Flask)  
- HTML, CSS, JavaScript  
- Render (Deployment)
